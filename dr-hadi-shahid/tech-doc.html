What Is This Website — In Plain English

Think of this website as a digital tribute book for Dr. A. Hadi Shahid — Founder of Alliott Hadi Shahid Chartered Accountants. Instead of a printed brochure, anyone in the world can visit www.drhadishahid.com and read his full biography, career timeline, credentials, press coverage, and watch his TV interviews.

Purpose
Personal legacy & professional profile website. Acts as the authoritative online presence for Dr. Shahid.
Audience
Clients, journalists, fellow professionals, investors, and family who want to learn about Dr. Shahid's career.
Technology
Pure HTML + CSS. No WordPress, no database, no monthly fees. Just files in a folder — simple and fast.
Cost to Host
Zero. Hosted on GitHub Pages for free. Only cost is the domain name renewal at GoDaddy (~$15/year).

How Hosting Works — Step by Step

When someone types www.drhadishahid.com in their browser, here is what happens in under 1 second:

Step 1 — DNS Lookup
Browser asks: "where is drhadishahid.com?"
GoDaddy DNS answers with GitHub's IP address (185.199.x.x)
Step 2 — HTTPS Handshake
Browser connects to GitHub's servers securely
Let's Encrypt certificate proves identity. All data is encrypted.
Step 3 — File Served
GitHub sends index.html to the browser
The browser reads the HTML and draws the page
Step 4 — Images Load
Photos are fetched from the same GitHub repo
Fonts load from Google Fonts CDN (fast, cached globally)

SEO — How Google Finds & Ranks the Site

SEO (Search Engine Optimisation) means making the site easy for Google to understand and rank highly when someone searches for "Dr. Hadi Shahid" or "Alliott Hadi Shahid Chartered Accountants".

FeatureWhat It IsWhy It Matters
Meta Title The text that appears in the browser tab and Google results First thing Google reads — includes Dr. Hadi Shahid's name
Meta Description The two-line summary shown under each Google result Tells users what the page is about before they click
Open Graph Hidden tags that control how the link looks on WhatsApp, LinkedIn, Twitter Portrait photo + title appear when link is shared
JSON-LD Schema Invisible structured data telling Google "this is a Person profile" Google may show a knowledge panel card in search results
Canonical URL Declares the "official" address of this page Prevents duplicate content penalties if the page appears at multiple URLs
Sitemap.xml A map of all pages submitted to Google Google discovers all pages faster, even without links to them
Robots.txt Instructions to search bots about what to index Allows all public pages to be indexed; points to sitemap
